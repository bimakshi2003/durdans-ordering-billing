# Durdans LIMS - Orders & Billing Module

Hospital Laboratory Information Management System - Frontend

## 🏥 Project Overview

Frontend module for managing laboratory test orders and billing for Durdans Hospital.

## 🚀 Tech Stack

- **Framework:** Next.js 14
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **UI Components:** shadcn/ui
- **Charts:** Recharts
- **Notifications:** Sonner

## 📦 Installation
```bash
npm install
```

## 🛠️ Development
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## 🏗️ Build
```bash
npm run build
```

## 📊 Features

- Dashboard with KPIs and analytics
- Test order creation and management
- Bills & payments tracking
- Payment history
- Revenue analysis with charts
- Tax invoices
- Role-based access control

## 🔒 Environment Variables

Create `.env.local`:
```env
NEXT_PUBLIC_API_URL=http://localhost:8080/api
NEXT_PUBLIC_WS_URL=ws://localhost:8080
```
